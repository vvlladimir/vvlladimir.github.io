Zope Managed Hosting

Servers are located at AT&T Data Centers anywhere in the AT&T Network. Strong physical access control coupled with industry-leading connectivity makes the AT&T Network an excellent choice for bussiness-critical hosting.

Customers enjoy access to ZOPE Replication Services ( ZRS ), a database replication technology that eliminates single points of failure in a server cluster.

Zope Corporation provides several types of hosting services:

  • Zope-tuned colocation services ( rack space, remote power menagement, network access )
  • Turnkey access to Zope Corporation Products
    • Zope4Media Broadcast
    • Zope4Media Print
    • Zope Registration Manager
    • Zope Replication Service
  • Shared and/or dedicated network servers/services
    • Load balancers
    • Cache/accelerator servers
    • LDAP servers
    • Database servers ( eg. mySQL, PostgreSQL, Oracle® )
    • Firewalls
  • Integrated ad serving solution
  • Network and network service monitoring and remediation services
  • Managed services for all Zope-based software delivered under Service Level Agreements

For more information about Zope Corporation's hosting services, please call (540) 361-1700 or email sales@zope.com.
